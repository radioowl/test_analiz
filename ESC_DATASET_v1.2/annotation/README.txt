# Описание

Файлы с аннотацией набора данных для распознавания голосовых команд руководителя маневров в рамках опытно-конструкторской работы "Пульт составителя эргономичный".

# Структура

Аннотация данных имеет следующую структуру:

|       Объект      |                  Описание                 |                                                                        Структура                                                               |
|:-----------------:|:-----------------------------------------:|:----------------------------------------------------------------------------------------------------------------------------------------------:|
| hr_bot_clear.json | Аннотация к набору данных hr_bot_clear    | audio_filepath (путь к аудиозаписи); id (id аудиозаписи); text (оригинальный текст речи); label (класс команды); attribute (атрибут команды)   |
| hr_bot_noise.json | Аннотация к набору данных hr_bot_noise    | audio_filepath (путь к аудиозаписи); id (id аудиозаписи); text (оригинальный текст речи); label (класс команды); attribute (атрибут команды)   |
| hr_bot_synt.json  | Аннотация к набору данных hr_bot_synt     | audio_filepath (путь к аудиозаписи); id (id аудиозаписи); text (оригинальный текст речи); label (класс команды); attribute (атрибут команды)   |
| luga.json         | Аннотация к набору данных luga            | audio_filepath (путь к аудиозаписи); id (id аудиозаписи); text (оригинальный текст речи); label (класс команды); attribute (атрибут команды)   |
| label2id.py       | Идентификация голосовых команд            | id (id команды); label (голосовая команда)                                                                                                     |
| extra             | ID спикеров в наборах данных hr_bot       | id (id аудиозаписи); cluster (id спикера)                                                                                                      |
