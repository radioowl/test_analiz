# Описание

Набор записей голосовых команд с выездов на станцию Лужская.

# Структура

Набор данных имеет следующую структуру:

| Директория | Формат записей |                                Описание                           |
|------------|:--------------:|:-----------------------------------------------------------------:|
| 03_07_2023 |       WAV      | Набор склеенных аудиозаписей с выезда на станцию Лужская 03.07.23 |
| 11_10_2023 |       WAV      | Набор голосовых команд с выезда на станцию Лужская 11.10.23       |
| 02_11_2023 |       WAV      | Набор голосовых команд с выезда на станцию Лужская 02.11.23       |
| 15_11_2023 |       WAV      | Набор голосовых команд с выезда на станцию Лужская 15.11.23       |
| 21_11_2023 |       WAV      | Набор голосовых команд с выезда на станцию Лужская 21.11.23       |
| noise      |       WAV      | Записи с примерами шума с Лужской (для генерации шума)            |
